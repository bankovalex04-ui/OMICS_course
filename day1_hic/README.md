# Day 1: Hi-C анализ

## Цель
Обработка Hi-C данных и построение контактных карт для образцов MoPh7, MoPh11, MoPh14, MoPh15.

## Результаты
- Получены `.hic` файлы для всех образцов.
- На картах видна сильная диагональ, что соответствует ожидаемому контакту соседних участков хромосом.
- Крупных межхромосомных перестроек не обнаружено.

## Файлы
- `scripts/run_hic_pipeline.sh` – bash-пайплайн для обработки сырых данных (FASTQ → .hic).
