# OMICS Course 

Этот репозиторий содержит выполненные практические задания по курсу OMICS (весна 2026).  
Все проекты объединены в одну структуру для удобства навигации.

## День 1: Hi-C анализ
**Задача**: обработать Hi-C данные и построить контактные карты для четырёх образцов (MoPh7, MoPh11, MoPh14, MoPh15) с целью выявления крупных хромосомных перестроек.  
**Конвейер**: подготовка референсного генома T2T (индексация BWA, создание chrom.sizes), обрезка адаптеров cutadapt, контроль качества FastQC, запуск Juicer для каждого образца с получением .hic файлов.  
**Результат**: полногеномные карты демонстрируют ожидаемый сильный диагональный паттерн контактов. Крупных межхромосомных перестроек не обнаружено. Карты имеют ограниченное разрешение, поэтому тонкие структуры (TADы, петли) требуют более глубокого покрытия для надёжной интерпретации.

## День 2: RNA-seq и CAGE анализ
**Задача**: получить треки покрытия для RNA-seq и CAGE, сравнить образцы в IGV и идентифицировать пример транскрипционно активного локуса.\
**Конвейер**: построение STAR индекса для T2T, выравнивание ридов STAR, генерация bigWig треков, визуализация в IGV. Для демонстрации преимуществ STAR перед BWA дополнительно сгенерирован трек выравнивания BWA для MoPh7.\
**Результат**: в локусе chr3:99,310,000–99,340,000 обнаружен совместный сигнал RNA-seq и CAGE, указывающий на активную транскрипцию. CAGE маркирует промоторную область, RNA-seq отражает покрытие транскрипта. Показано, что STAR корректно учитывает сплайсинг, в отличие от BWA, что делает его предпочтительным инструментом для RNA-seq данных.

## День 3: ChIP-seq анализ
**Задача**: обработать ChIP-seq данные для гистоновых меток (H3K9me3, H3K27Ac, H2A119Ub) и CTCF, сгенерировать треки покрытия и пики, выполнить визуализацию в IGV.\
**Конвейер**: обрезка адаптеров cutadapt, выравнивание BWA MEM на T2T, сортировка и индексация samtools, фильтрация по MAPQ≥30, генерация треков покрытия q30, пиковый вызов MACS3 с генерацией треков обогащения.\
**Ограничение**: полный анализ ресурсоёмок, поэтому использована подвыборка 1M парных ридов на образец.\
**Результат**: для H3K27Ac и CTCF получены непустые narrowPeak файлы; для H3K9me3 и H2A119Ub broadPeak файлы пустые, однако треки покрытия и обогащения MACS3 валидны. Пример региона визуализации: chr1:144,390,000–144,430,000.

## День 4: WGBS (анализ метилирования ДНК)
**Задача**: проанализировать готовый результат Bismark для MoPh7, рассчитать метилирование CpG, создать треки и интегрировать данные с результатами ChIP-seq из Дня 3.\
**Конвейер**: повторное выравнивание не проводилось; использованы готовые данные Bismark. Рассчитаны охват, бета-значение и M-значение. Сгенерированы треки MoPh7_beta_methylation.bw, MoPh7_m_value.bw, MoPh7_coverage.bw. Проведена интеграция: метилирование анализировалось вокруг пиков H3K27Ac, вычислены корреляции с ChIP-seq сигналом.\
**Результат**: пики H3K27Ac, маркирующие активные регуляторные области, демонстрируют локальное снижение метилирования вблизи центров пиков. Анализ корреляции CpG-сайтов в масштабах всего генома показывает слабую отрицательную связь между метилированием и сигналом H3K27Ac (Пирсон –0.152, Спирмен –0.070) и H3K9me3 (Пирсон –0.048, Спирмен –0.049). Пример региона визуализации: chr1:16,362,664–16,402,664.
