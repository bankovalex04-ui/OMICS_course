# OMICS Course – Объединённый репозиторий

Этот репозиторий содержит выполненные практические задания по курсу OMICS (весна 2026).  
Все проекты объединены в одну структуру для удобства навигации.

## Структура

- [`day1_hic/`](day1_hic/) – Hi-C анализ (обработка данных, контактные карты).
- [`day2_rnaseq_cage/`](day2_rnaseq_cage/) – RNA-seq и CAGE анализ (выравнивание, треки покрытия).
- [`day3_chipseq/`](day3_chipseq/) – ChIP-seq анализ (пики, визуализация).
- [`day4_wgbs/`](day4_wgbs/) – WGBS анализ (метилирование ДНК).

## Важное замечание
Большие файлы (BAM, FASTQ, bigWig, .hic) не включены в репозиторий.  
Все исходные данные доступны по ссылкам из курса.
