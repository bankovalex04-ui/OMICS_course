# Day 3: ChIP-seq анализ

## Цель
Обработка ChIP-seq данных для гистоновых меток H3K9me3, H3K27Ac, H2A119Ub и CTCF, поиск пиков и визуализация в IGV.

## Результаты
- Выявлены пики обогащения, особенно сильные для CTCF.
- Пример локуса: chr20:33.44–33.45 Mb.
