# Day 2: RNA-seq и CAGE анализ

## Цель
Получить треки покрытия для RNA-seq и CAGE, сравнить их в IGV и найти пример транскрипционно активного локуса.

## Результаты
- В локусе chr3:99,310,000–99,340,000 обнаружен совместный сигнал RNA-seq и CAGE.
- Показано, что STAR лучше подходит для выравнивания RNA-seq, чем BWA, так как учитывает сплайсинг.
